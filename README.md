# akibjawad.github.io

Personal academic portfolio and resume site for **Akib Jawad Nafis**.

Built with Next.js, [shadcn/ui](https://ui.shadcn.com/), and [Magic UI](https://magicui.design/), deployed on GitHub Pages.

Based on the open-source portfolio template by [Dillion Verma](https://github.com/dillionverma/portfolio), licensed under the [MIT License](https://github.com/dillionverma/portfolio/blob/main/LICENSE.md).
