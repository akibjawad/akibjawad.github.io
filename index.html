Publications

On constructing contention aware connected dominating sets for inter-connectivity among internet of things devices

This was an attempt to create contention aware connected dominating set for efficient broadcasting in wireless adhoc networks. Creating a minimum connected dominating set will not always assure contention free network. With more contention there is more bandwidth loss. This bandwidth loss is huge when we are broadcasting in a wireless adhoc network. In this article, we proposed a hybrid algorithm which will help to reduce contention. For performance comparison we simulated other available algorithms with the help of NS-2 simulator along with our hybrid algorithm.

Academic Projects

Movie-Recommendation sentiment analysis

Classification of a movie recommendation to any of these 5 (very bad, bad, average, good, very good) catagories was the primary goal of this project. Achieving this entails analyzing sentiment of sentences correctly. A program was written on top of keras IMDB movie database which uses LSTM to find out relationships among words in a sentence. Performance comparison was done by changing auto-encoder layers & by modifying cross-validation methods.

Writing an operating system with Nachos

An attempt to build a complete operating system which can compile any C/C++ code and run program in a simulated machine. This operating system includes multithread support along with paging mechanism to run system in a ultra low memory configuration. Multithread support is managed with pThread library in c++, NFU algorithm was used for paging mechanism and Banker's Algorithm was used for resource allocation. These parts are written on top of NachOS.

Finding optimum path to avoid traffic by analyzing crowd sourced data.

With the increase of smartphone usage while stuck in traffic, collecting traffic info from the crowd itself is an enticing idea. Avoiding traditional automatic location tracking gives one opportunity to avoid security violation and access to more structured data. Hence this was an attempt to find the road with least traffic. In this two parts project, part one(Frontend) was built with google maps API for android phones while part two(Backend) was built with django(Python) for data analysis.

An attempt to create a GCC clone with Flex and Bison

This experiment was to create a compiler which can compile a large subset C/C++ source code and generate assembly code. Symbol table was programmed in C++. Flex was used to create lexical analyser while semantic analysis was done with the help of Bison. Merging these three parts this system can generate assembly code from C/C++ source code in an asm file format. Correctness of the programme was validated by running the output through an emu8086 assembler.
